# Data

이 디렉터리는 브라우저 앱이 런타임 외부 API 없이 사용할 정적 기준 데이터를 담습니다.

데이터는 아래 명령으로 다시 생성합니다.

```sh
node scripts/update-data.mjs
```

## 파일

- `sources.json`
  - 수집 출처, 접근 시각, 공식 전환율 같은 메타데이터입니다.
- `price-indexes.json`
  - 화폐별 연도 단위 물가지수 또는 구매력 계수입니다.
  - `provisionalYears`가 있는 연도는 확정 연간 평균이 아닌 부분 연도 또는 최신 월 기준 값입니다.
- `exchange-rates.json`
  - USD 1달러당 현지 통화 단위의 연평균 환율입니다.
  - 2026년 환율은 ECB 일별 기준환율의 연초 이후 평균값이므로 확정 연평균이 아닙니다.

## 현재 범위

- GBP 물가지수: 1209-2026
- FRF 물가지수: 1901-2025
- EUR 물가지수: 1996-2025, 실제 계산에서는 1999년 이후 사용
- USD 물가지수: 1913-2026
- RUB_IMP 브리지 지수: 1880-1913 입력, 1998-2026 목표 연도
- SUR 브리지 지수: 1961-1991 입력, 1998-2026 목표 연도
- RUR/RUB 물가지수: 1992-2025
  - RUR 입력: 1992-1997
  - RUB 입력: 1998-2025
- GBP/KRW/USD 환율: 1960-2026
- EUR 환율: 1999-2026
- FRF 환율: 1960-1998
- RUR 환율: 1993, 1996-1997
- RUB 환율: 1998-2024

제정 러시아 루블(`RUB_IMP`)과 소련 루블(`SUR`)은 Historicalstatistics.org의 Historical Currency Converter를 사용한 실험적 브리지 지수입니다. 러시아 현지 CPI가 아니라 스웨덴 소비재 구매력 비교를 통해 목표 연도 미국 달러 감각으로 넘긴 뒤, 원화가 필요하면 USD/KRW 환율을 적용합니다.

러시아 구루블(`RUR`)은 1998년 화폐개혁 전 금액을 뜻합니다. 1998년 이후 현행 루블(`RUB`)로 넘길 때는 `1 RUB = 1,000 RUR` 전환율을 적용합니다. 1914-1960 구간은 자료 공백과 1920년대/1947년 화폐개혁 해석 문제 때문에 현재 범위에 포함하지 않습니다.

이 데이터는 독서 중 시대상 이해를 위한 근사 계산용입니다. 정교한 회계, 투자, 법률 판단에 쓰지 않습니다.
